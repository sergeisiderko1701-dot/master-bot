# Master Service Bot — UX redesign

У цій версії я оновив візуальну частину бота без зміни основної логіки:
- більш охайні картки заявки і майстра
- нові тексти старту, меню і підтримки
- покращений UI чату
- окрема reply-клавіатура для чату
- чистіші повідомлення про офери

## Запуск

1. Скопіюй `.env.example` у `.env`
2. Заповни:
   - `BOT_TOKEN`
   - `ADMIN_ID`
   - `DATABASE_URL`
3. Встанови залежності:
   ```bash
   pip install -r requirements.txt
   ```
4. Запусти:
   ```bash
   python app.py
   ```

## Важливо
- Використай новий Telegram token.
- Це готовий код проекту: можна копіювати в окрему папку і запускати.
